分步实施：
阿斯蒂芬 
1、完成上述添加、查找（课程编号、课程名称、课程性质等）、修改等功能；

2、学生选修课程。

2、利用结构体数组实现信息的数据结构设计或利用单向链表实现信息的数据结构设计；

3、界面友好（良好的人机交互），程序加必要的注释。
